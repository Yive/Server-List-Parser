# Server-List-Parser
Parser for minecraft server lists.


### Usage:
```perl parse.pl [valid server list]```

### Valid Server Lists:
```minecraftservers.org
minecraft-server-list.com
minecraft-index.com
minecraft-server.net
minelist.net
minecraftforum.net
minepick.com
serveurs-minecraft.com
minecraftpeservers.org
```
